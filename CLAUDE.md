# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Overview

This is a **chezmoi-managed dotfiles** setup for macOS. The chezmoi source directory lives at `~/.local/share/chezmoi` and manages config files at their real paths (e.g. `~/.zshrc`, `~/.config/aerospace/aerospace.toml`).

All dotfiles operations use chezmoi:
```bash
chezmoi add ~/.config/newapp/config.yml   # track a new file
chezmoi edit ~/.zshrc                     # edit via chezmoi
chezmoi apply                             # apply changes to home dir
chezmoi diff                              # see what would change
chezmoi cd                                # cd into source directory
```

The `cz` alias is available for `chezmoi`.

## Repository Structure

### Shell
- `~/.zshrc` - Zsh config (PATH, plugins, tool init)
- `~/.zsh_aliases` - Shell aliases
- `~/.tmux.conf` - tmux config (vi keys, TPM)
- `~/.gitconfig` - Git user config

### Window Management
- `~/.config/aerospace/aerospace.toml` - AeroSpace tiling WM config
- `~/.config/karabiner/karabiner.json` - Karabiner-Elements config (generated by Goku)
- `~/.config/goku/karabiner.edn` - Goku source (edit this, not karabiner.json directly)
- `~/.config/sketchybar/` - SketchyBar status bar (Lua config)

### Terminal
- `~/.config/ghostty/config` - Ghostty terminal config
- `~/.config/ghostty/shaders/` - Ghostty custom shaders
- `~/.config/starship.toml` - Starship prompt config (Gruvbox Dark)
- `~/.config/lazygit/config.yml` - Lazygit config (Catppuccin theme)

### Tools
- `~/.config/brewfile/Brewfile` - Homebrew packages, casks, and Mac App Store apps
- `~/.config/gh/config.yml` - GitHub CLI config (NOT hosts.yml - contains auth)
- `~/.ssh/config` - SSH config (NOT keys)
- `~/.config/thefuck/settings.py` - thefuck settings
- `~/.config/git/ignore` - Global gitignore

### IDE
- `~/.config/jetbrains-keymaps/macOS_mod.xml` - Canonical JetBrains keymap (distributed to all IDEs via setup.sh)

### macOS
- `~/Library/Preferences/com.apple.HIToolbox.plist` - Input sources config

### Scripts
- `~/scripts/merge_pr.sh` - Interactive PR merge helper using `gh` CLI
- `~/scripts/script_manager.sh` - CLI for managing shell aliases and scripts
- `~/scripts/setup.sh` - Bootstrap script for new Mac setup

### Repo Meta
- `~/.gitignore` - Allowlist gitignore (lives at $HOME root)
- `~/CLAUDE.md` - This file

## Key Conventions

- **chezmoi**: Use `chezmoi add` to track new files, `chezmoi apply` to apply changes. The source repo at `~/.local/share/chezmoi` is pushed to GitHub.
- **Karabiner via Goku**: Edit `~/.config/goku/karabiner.edn`, then run `goku` to regenerate `karabiner.json`. Never edit `karabiner.json` directly.
- **AeroSpace**: Edit `~/.config/aerospace/aerospace.toml` directly. Super key = ctrl+alt+cmd. Service mode via super+semicolon.
- **SketchyBar**: Lua-based config in `~/.config/sketchybar/`. Rebuild helpers with `cd ~/.config/sketchybar/helpers && bash install.sh`.
- **JetBrains keymaps**: Edit `~/.config/jetbrains-keymaps/macOS_mod.xml` as the single source of truth. Run `scripts/setup.sh` (section 5) or manually copy to `~/Library/Application Support/JetBrains/*/keymaps/`.
- **Shell scripts**: Use `#!/bin/bash`, include input validation, use `gh` CLI for GitHub operations.
- **No secrets**: Never track auth tokens, SSH keys, or API keys.
- **No build system**: This is a config-only repo with no build, lint, or test commands.

## Common Operations

```bash
# Check what chezmoi would change
chezmoi diff

# Add a new config file
chezmoi add ~/.config/newapp/config.yml

# Edit and apply
chezmoi edit ~/.zshrc
chezmoi apply

# Commit and push chezmoi source
chezmoi cd
git add -A && git commit -m "description" && git push

# Update Brewfile after installing new packages
brew bundle dump --file=~/.config/brewfile/Brewfile --force

# Regenerate Karabiner config
goku
```
